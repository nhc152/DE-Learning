- **Công cụ DB Client nghiệp dư:** VS Code SQLTools xài nhẹ để test thì tốt, nhưng thao tác với warehouse cần ERD diagram, Execution Plan Analysis hay import file csv hàng triệu dòng thì sẽ sập. Cần có DBeaver hoặc DataGrip.
