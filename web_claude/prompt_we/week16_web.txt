NỘI DUNG CẦN BAO PHỦ:
- Khi nào pandas / SQL local không đủ và vì sao Spark xuất hiện
- Spark như distributed compute engine, không chỉ là “pandas lớn hơn”
- Spark architecture:
  - Driver
  - Executor
  - Cluster manager
  - Stage
  - Task
  - Shuffle
  - narrow vs wide transformation
- RDD vs DataFrame vs Dataset:
  - khi nào dùng DataFrame
  - vì sao RDD ít dùng trong DE thông thường
  - trade-off abstraction vs control
- Lazy evaluation:
  - transformations không chạy ngay
  - actions mới trigger execution
  - lineage
  - optimizer mindset
- Join strategies:
  - broadcast join
  - sort-merge join
  - shuffle hash join
  - khi nào mỗi loại phù hợp
  - join skew và vấn đề thực tế
- Partitioning:
  - repartition vs coalesce
  - partitionBy khi ghi file
  - số lượng partition ảnh hưởng hiệu năng
  - small files problem
- Bucketing / sorting / cache:
  - khi nào bucket hữu ích
  - sortWithinPartitions để giảm shuffle
  - khi nào cache/persist hợp lý
  - anti-pattern khi cache bừa
- Spark UI:
  - job / stage / task view
  - shuffle read / write
  - skew detection
  - bottleneck analysis
- Unit testing:
  - test transformation functions với fake DataFrame
  - assert logic output
  - tách business logic khỏi IO
- Artifact spark_jobs/ gồm:
  - broadcast_join_job.py
  - bucket_join_job.py
  - aggregation_optimized.py
  - sort_within_partitions.py
  - tests/test_transform_functions.py
  - README.md mô tả job, optimization, test strategy
