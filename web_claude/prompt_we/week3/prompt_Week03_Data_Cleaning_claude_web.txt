Các module bắt buộc:
- W3.01 – Tổng Quan & Tidy Data Principles
- W3.02 – Data Profiling: Hiểu Data Trước Khi Làm
- W3.03 – Xử Lý Missing Values
- W3.04 – Deduplication Strategies

NỘI DUNG CẦN BAO PHỦ:
- Tidy Data principles theo tinh thần Hadley Wickham
- Data profiling trước khi load
- Missing values: identify, analyze, imputation, dropping, business trade-off
- Duplicates: exact duplicate, business-key duplicate, window function dedup pattern
- Outliers: detect, validate, cap/remove/flag, phân biệt outlier thật và data error
- Wrong data types, schema drift, validation rules
- Defensive programming: assert, schema validation, fail fast
- pandera cho schema + business rules validation
- before/after comparison của cleaning pipeline
- artifact week3_data_cleaning/ gồm:
  - profiling_report.ipynb
  - cleaning_pipeline.py
  - validation_rules.py
  - before_after_comparison.md
