NỘI DUNG CẦN BAO PHỦ:
- Vì sao DE cần cloud:
  - local chỉ đủ để học nền tảng
  - production cần storage, IAM, monitoring, managed services, scalability
  - move từ local stack sang production-like environment
- S3 cho Data Lake:
  - bucket strategy: raw / staging / curated / archive
  - object storage mindset
  - partitioning theo date/domain/source
  - naming convention
  - lifecycle policy
  - versioning
  - encryption
  - small files problem
  - anti-pattern khi dùng S3
- IAM:
  - user vs group vs role
  - IAM Role cho EC2 / Airflow / Lambda / Glue / CI/CD
  - inline policy vs managed policy
  - least privilege
  - cross-service access cho data pipeline
  - ví dụ policy chỉ cho phép đọc raw và ghi curated
- RDS/Aurora:
  - khi nào dùng RDS PostgreSQL
  - Aurora khác gì RDS thường
  - use cases cho Airflow metadata DB
  - operational database vs analytical warehouse
  - backup, Multi-AZ, connection basics
- Athena vs Redshift:
  - Athena query trực tiếp trên S3
  - Redshift như data warehouse managed
  - khi nào dùng Athena
  - khi nào dùng Redshift
  - cost/performance trade-off
  - external tables / schema-on-read mindset
- CloudWatch:
  - log groups, metrics, alarms
  - log aggregation cho jobs
  - giám sát pipeline runs
  - alerting basics
  - ví dụ metric cho failed jobs / latency / freshness
- Glue Data Catalog:
  - catalog tables
  - crawler concept
  - schema metadata
  - relationship giữa S3 + Athena + Glue
  - khi nào crawler hữu ích, khi nào nên define schema chủ động
- Terraform / IaC:
  - tại sao không nên click tay toàn bộ trên console
  - tổ chức thư mục terraform/
  - s3_buckets.tf
  - iam_roles.tf
  - rds_metadata.tf
  - variables.tf
  - outputs.tf
  - README mô tả kiến trúc, cost estimate, security notes
- Cost & security:
  - least privilege
  - encryption at rest / in transit
  - bucket public access block
  - tagging resources
  - cost estimate mindset
  - avoid over-provisioning
  - production design trade-offs
- Artifact cloud_setup/ gồm:
  - terraform/s3_buckets.tf
  - terraform/iam_roles.tf
  - terraform/rds_metadata.tf
  - terraform/variables.tf
  - terraform/outputs.tf
  - architecture.md
  - README.md mô tả mục tiêu, kiến trúc, cost estimate, security notes, cách apply
