# DE-LEARNING Project Rules

## Project

Interactive HTML tutorial series for learning Data Engineering.

## Primary Goal

Build high-quality weekly tutorial files (`WeekXX_*.html`) that are:

* practical
* visually consistent
* beginner-friendly
* production-oriented
* reusable long-term

---

## Core Principles

* Do not overwrite strong existing content blindly
* Improve weak areas first
* Preserve consistency across all weeks
* Prefer practical examples over abstract theory
* Prefer teaching clarity over fancy wording
* Prefer realistic production scenarios
* Keep outputs maintainable

---

## Default Output Target

Unless asked otherwise:

* create or update only target `WeekXX_*.html`

Do not create extra files unless requested.

---

## Working Mode

Assume this project is a long-term handbook system.

Always optimize for:

* same tutorial family feeling
* consistent UI/UX
* scalable future expansion
* useful portfolio quality
