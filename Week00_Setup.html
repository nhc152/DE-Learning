Workspace settings gợi ý

  "editor.formatOnSave": true,

Lệnh setup cơ bản

Terminal
python --version
pip --version
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activate
python -m pip install --upgrade pip
pip install pandas sqlalchemy psycopg2-binary

Checklist xác minh

  • Python báo 3.11+
  • pip chạy trong .venv
  • Import pandas không lỗi
  • Có file requirements.txt để khóa package
Nếu cài Python xong nhưng terminal vẫn ra bản cũ, kiểm tra PATH hoặc mở terminal mới.

Chiến lược quản lý dependency cho người mới

  1. Cố định phiên bản các package cốt lõi bằng requirements.txt.
  2. Khi thêm package mới, cài xong thì cập nhật lại file lock ngay.
  3. Nếu project lỗi dependency khó sửa, tạo lại .venv từ đầu để làm sạch môi trường.
Chuỗi lệnh refresh môi trường
# khi cần làm sạch môi trường
# Windows: rmdir /s /q .venv
# macOS/Linux: rm -rf .venv
python -m venv .venv
# activate theo hệ điều hành rồi cài lại
pip install -r requirements.txt

Lệnh Git bắt buộc

Git CLI
git config --global user.name "Your Name"

ssh-keygen -t ed25519 -C "you@example.com"
git init
git add .
git commit -m "setup: initialize week00 foundation"
git branch -M main
git remote add origin git@github.com:<user>/de-roadmap-artifacts.git
git push -u origin main

Conventional commit gợi ý

  • setup: thay đổi môi trường, cấu hình
  • feat: thêm tính năng/code mới
  • fix: sửa lỗi
  • docs: cập nhật ghi chú học tập
Nếu Permission denied (publickey), kiểm tra lại SSH key đã add vào GitHub và đang dùng remote SSH.

Workflow commit thực tế cho người học DE

Tình huốngNên commit thế nàoVí dụ message
Vừa cấu hình editorCommit riêng cấu hìnhsetup: configure vscode workspace settings
Vừa thêm script verify PythonCommit riêng scriptsetup: add python environment verification script
Vừa thêm SQL testCommit riêng SQLsetup: add postgres smoke test query

Checklist trước khi push

  • Đã chạy nhanh lệnh chính (python/sql/docker) ít nhất 1 lần
  • Không có file nhạy cảm trong stage (.env, password thật)
  • Commit message mô tả đúng mục đích thay đổi
  • Repo remote đúng URL mong muốn

Checklist xử lý lỗi nhanh

  • Port conflict: đổi host port trong compose.
  • Container exit ngay: đọc docker logs để biết nguyên nhân thật.

  • Volume là lớp dữ liệu bền, cần giữ khi học pipeline nhiều ngày.
